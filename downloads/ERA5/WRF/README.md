## Initialize WRF with ERA5 Data


### 1) Run WRF with ERA5 model level data (138 vertical levels)   

**Scripts:**
- `getERA5-WRF-ml.py` retrieves model level data   
- `getERA5-WRF-sfc.py` retrieves surface data   

**Tutorial:** https://dreambooker.site/2018/04/20/Initializing-the-WRF-model-with-ERA5/ 


### 2) Run WRF with ERA5 pressure level data (38 vertical levels)   

**Scripts:**
- `getERA5-WRF-pl.py` retrieves pressure level data  
- `getERA5-WRF-sl.py` retrieves single/surface level data  

**Tutorial:** https://dreambooker.site/2019/10/03/Initializing-the-WRF-model-with-ERA5-pressure-level/
